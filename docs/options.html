rowOption 选择记录行的参数
object | false {
type: 'checkbox',
showSelectAll: true,
inputName: "row.id"
}
如果为 false，就不显示全选按钮，也没有click点击某行的特性
type string 'checkbox' 取值范围：checkbox，radio, none
showSelectAll boolean true 控制是否显示全选按钮，如果type=radio或none，不会显示全选按钮
inputName string 'row.id' table里每行里的checkbox或radio的name，如果type=none，会忽略此值
rowSelector string '> tbody > tr' 数据行 selector

pageOption 分页widget
object | false 如果为 false，那么就不显示分页widget
否则必须是包含是以下属性的object: currentPage, totalPages, totalRows
rowsPerPageOptions string '20,50,100,200,500,1000' 每页多少条的选项
currentPage int 当前第几页, 如果没填则不显示分页widget
rowsPerPage int rowsPerPageOptions[0] 每页多少条
totalPages int 总页数, 如果没填则不显示分页widget
totalRows int 总行数, 如果没填则不显示分页widget

sortOption 排序状态
object sortOption: {} 设置排序状态
sortOption: {
'A': 'desc',
'B: 'asc'
}
表头排序状态 'sortByName': 'order' 在table的某个元素上设置data-table-sort-by="A" 就能激活排序功能

设置'A'的排序状态 sortOption: { 'A': 'desc' }

menus 菜单按钮列表
array of MenuItemDefinition menus: [] 菜单按钮配置列表,

MenuItemDefinition有4种:
1.按钮, 2.Simple, 3.Simple
4.将以上三种分到一组里(放到一个数组里)

MenuItemDefinition - 按钮
object 菜单按钮的配置
{
title: ...,
icon: ...,
btnClass: ...,
tooltip: ...,
callback: ...
}
title string 非必填，按钮标题
icon string 非必填，可以是 Fontawesome, Glyphicons的图标字体，比如 fa fa-calendar
btnClass string 非必填，覆盖最上面btnClass的设置
tooltip string 非必填，鼠标悬浮时显示的 tooltip
callback function 非必填，点击按钮后触发的回调函数，比如这样
function () {
var semiAutoTable = this; // this指向semiAutoTable对象
}

MenuItemDefinition - Simple
object Simple的配置
{
title: ..., (非必填)
icon: ..., (非必填)
btnClass: ..., (非必填)
tooltip: ..., (非必填)
dropdowns: ...
}
dropdowns array of Simple项 Simple项的数组，Simple项的定义看下面
Simple项 object | 'divider' 菜单项:
{
title: ...,
icon: ..., (非必填)
callback: ...,
}
分隔线: 'divider'
